# Editing Checklist

Use this checklist before finalizing a report.

## Structure
- Is the document type clear?
- Does the opening answer the real question?
- Are sections in the right order?
- Are there duplicated ideas that should be merged?

## Clarity
- Are conclusions explicit?
- Are key distinctions clear?
- Is uncertainty labeled honestly?
- Is jargon only used where it helps?

## Utility
- Can the user act on the result?
- Is there a next step?
- Would this save time later?
- Is the document reusable?

## Compression
- Can any paragraph be cut without loss?
- Can a list replace a rambling block?
- Is there unnecessary throat-clearing or filler?

## Final quality bar
- Does it read like a final deliverable?
- Would you send this without apology?
